Evaluating Functions

Understanding how to evaluate functions by replacing variables with given values.

Intro

Evaluating a function involves substituting the variable with a specific value from the function's domain:

  • To represent evaluating F at a for some a in the domain of F, we write F(a).

Evaluating Functions

Here's how to evaluate functions with examples:

  1. Given function: f(x) = 2x + 1
  2. Evaluating f(x) at x = 1.5:
    • Substituting 1.5 for x: f(1.5) = 2(1.5) + 1
    • Performing the operations: f(1.5) = 3 + 1 = 4
    • Therefore, f(1.5) = 4.
  3. Given function: q(x) = x^2 - 2x + 2
  4. Evaluating q(x) at x = 2:
    • Substituting 2 for x: q(2) = (2)^2 - 2(2) + 2
    • Performing the operations: q(2) = 4 - 4 + 2 = 2
    • Therefore, q(2) = 2.
  5. Given function: g(x) = √(x + 1)
  6. Evaluating g(x) at x = 1.5:
    • Substituting 1.5 for x: g(1.5) = √(1.5 + 1)
    • Performing the operations: g(1.5) = √(2.5) ≈ 1.58
    • Therefore, g(1.5) ≈ 1.58.

Examples

Further examples of evaluating functions:

  • f(x) = 2x + 1:
    • For x = 1.5, f(x) = 8.
  • Evaluating functions at x = 3:
    • f(x) = x - 3, f(3) = 0.
    • g(x) = x^2 - 3x + 5, g(3) = 5.
    • h(x) = ∛(x^3 + x + 3), h(3) = ∛33.
    • p(x) = x^2 + 1/x - 4, p(3) = -10.

Function Restrictions

Understanding function restrictions and undefined values:

  • For g(x) = √(x - 1), the domain is x ≥ 1.
  • For r(x) = 1/x, the domain is x ≠ 0.

Conclusion

Mastering function evaluation helps in understanding function behaviors and their applications.
