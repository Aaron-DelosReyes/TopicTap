Understanding Functions and Relations

Let's dive into the diverse ways of representing functions and relations:

Introduction

Functions and relations can be depicted using:

  • Sets of ordered pairs
  • Mapping diagrams
  • Tables of values
  • Graphs

Mapping Diagram

A mapping diagram visually represents the relationship between x and y values:

  • If each domain element is paired with exactly one range value, it illustrates a function.
  • If an x value is paired with more than one y value, it does not represent a function.

Table of Values

A table of values provides a tabular representation of the function:

  • If each x value corresponds to exactly one y value, the table represents a function.

Graphs

Graphs visually depict relations and functions:

  • Use the vertical line test to determine if a graph represents a function.

Vertical Line Test

The vertical line test helps identify if a graph represents a function:

  • If every vertical line intersects the graph at most once, it signifies a function.
  • If any vertical line intersects the graph multiple times, it does not represent a function.
